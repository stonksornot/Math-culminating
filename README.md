This is the math culminating project

It uses replit for server hosting, and socket.io for backend/frontend communication.

index.js is the backend logic
Admin.html is the start button
Board.html acts as the leaderboard
Player.html is the player screen
index.html lets you choose between board and player. This is where players join

Features:
* Real time multiplayer
* 25 math problems
* Sharable by link

Please note this is not a perfect project. I made this in 3 days because that's how much time I had for the culminating.
